# CalmMind OCD – OCD Awareness & Support Website

A modern, accessible, and privacy-focused single-page website for OCD (Obsessive–Compulsive Disorder) awareness, education, and self-reflection. Features a self-assessment quiz, coping strategies, resources, and an integrated Botpress chatbot for supportive conversation.

## Features
- Clean, calming, and professional UI (HTML5, CSS3, Vanilla JS)
- Responsive and accessible design for all devices
- Self-assessment quiz (no data stored)
- Coping strategies and FAQ
- Resources and emergency help section
- Botpress chatbot (button in bottom left corner)
- No user data or chat is stored on the server

## Getting Started
1. **Clone or download this repository.**
2. **Open `index.html` in any modern web browser.**
   - No build step or server required for basic use.
   - For live reload, use an extension like Live Server in VS Code.

## File Structure
- `index.html` – Main HTML file with all sections
- `style.css` – All styles (no frameworks)
- `script.js` – All interactivity (quiz, navigation, chatbot trigger, etc.)

## Customization
- **Text content:** Edit in `index.html`.
- **Colors & fonts:** Edit CSS variables at the top of `style.css`.
- **Quiz questions/logic:** Edit in `script.js` (see quiz logic section).
- **Botpress chatbot:** Configured via Botpress Cloud; see their docs for customization.

## Safety Notice
This site is for education and self-reflection only. It does not provide diagnosis, therapy, or emergency support. For real help, contact a mental health professional or emergency services.
